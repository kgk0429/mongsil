# AI 기반 영어유치원 비교 서비스 - 랜딩페이지

수요 검증용 랜딩페이지 (AIDA 프레임워크 기반)

## 📁 파일 구조

- `index.html` - 메인 HTML 파일
- `styles.css` - 스타일시트
- `script.js` - JavaScript 인터랙션
- `start_server.py` - 로컬 서버 실행 스크립트 (선택사항)

## 🚀 로컬에서 실행하기

### 방법 1: Python 서버 사용
```bash
python3 start_server.py
```
또는
```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속

### 방법 2: 파일 직접 열기
`index.html` 파일을 브라우저에서 직접 열기

## 📋 섹션 구성

1. **Hero Section (Attention)** - 강렬한 헤드라인
2. **문제 공감 Section (Interest)** - 부모들의 공감대 형성
3. **기능 소개 Section (Desire)** - 3가지 핵심 가치
4. **예시 및 인사이트 Section (Desire 강화)** - 구체적 이점
5. **CTA Section (Action)** - 행동 유도 버튼

## 🎨 주요 특징

- 반응형 디자인 (모바일/태블릿/데스크톱)
- 모던한 UI/UX
- 부드러운 애니메이션 효과
- AIDA 마케팅 프레임워크 적용
